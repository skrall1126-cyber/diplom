# 🚀 Хурдан эхлүүлэх заавар

## ✅ Одоогийн байдал

Таны систем бэлэн болсон! Зөвхөн админ хэрэглэгчдийг өгөгдлийн санд нэмэх шаардлагатай.

## 📋 Хийх ёстой зүйлс

### 1️⃣ Админ хэрэглэгчдийг үүсгэх (5 минут)

1. **Supabase руу нэвтрэх:**
   - https://supabase.com/dashboard орно уу
   - Төслөө сонгоно уу

2. **SQL Editor руу орох:**
   - Зүүн цэснээс "SQL Editor" сонгоно уу
   - Эсвэл: https://supabase.com/dashboard/project/xdywicnxzgtstrslhvcu/sql

3. **SQL кодыг ажиллуулах:**
   ```bash
   # scripts/insert-users.sql файлын агуулгыг хуулаад SQL Editor дээр буулгаж Run дарна уу
   ```

4. **Нэвтрэх:**
   - http://localhost:3000/admin/login руу орно уу
   - Хэрэглэгчийн нэр: `admin`
   - Нууц үг: `admin123`

### 2️⃣ Серверийг эхлүүлэх

```bash
cd indra-cyber-home/indra-home
npm run dev
```

Сервер http://localhost:3000 дээр ажиллана.

## 🔧 Тохиргоо

### Environment файлууд:
- ✅ `.env.local` - Бүрэн тохируулсан
- ✅ `SUPABASE_URL` - Холбогдсон
- ✅ `SUPABASE_ANON_KEY` - Тохируулсан
- ✅ `SUPABASE_SERVICE_ROLE_KEY` - Тохируулсан
- ✅ `JWT_SECRET` - Тохируулсан

### Өгөгдлийн сан:
- ✅ Supabase холбогдсон
- ⏳ Хэрэглэгчдийг нэмэх шаардлагатай (дээрх SQL-ийг ажиллуулна уу)

## ❓ "Invalid API key" алдаа гарч байна уу?

Энэ алдаа гарах шалтгаан:
- Хэрэглэгчид өгөгдлийн санд байхгүй байна

Шийдэл:
1. `scripts/insert-users.sql` файлыг нээнэ үү
2. Агуулгыг хуулна уу
3. Supabase SQL Editor дээр ажиллуулна уу
4. Дахин нэвтэрч үзнэ үү
