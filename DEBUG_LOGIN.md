# 🐛 Нэвтрэх асуудал шийдэх

## Асуудал: Удаан ачаалагдаж, admin page руу шилжихгүй байна

### Шалгах зүйлс:

#### 1️⃣ Browser Console шалгах

**Chrome/Edge:**
- `F12` дарж Developer Tools нээнэ үү
- **Console** tab сонгоно уу
- Нэвтрэх товч дарна уу
- Алдаа байгаа эсэхийг харна уу

**Харах ёстой log-ууд:**
```
🔐 Нэвтрэх оролдлого: admin
📡 Response: {user: {...}, token: "..."}
✅ Амжилттай нэвтэрлээ: admin
🚀 Чиглүүлж байна: /admin/dashboard
```

#### 2️⃣ Network tab шалгах

**Developer Tools дээр:**
- **Network** tab сонгоно уу
- Нэвтрэх товч дарна уу
- `/api/auth/login` хүсэлтийг олно уу
- Status: `200 OK` байх ёстой

**Хэрэв 401 эсвэл 500 бол:**
- Response tab дээр алдааг харна уу
- Console tab дээр дэлгэрэнгүй мэдээлэл харна уу

#### 3️⃣ Сервер ажиллаж байгаа эсэхийг шалгах

Terminal дээр:
```bash
# Сервер ажиллаж байгаа эсэхийг шалгах
node scripts/test-login.js
```

Амжилттай бол:
```
✅ Нэвтрэлт амжилттай!
👤 Хэрэглэгч: admin
```

#### 4️⃣ LocalStorage шалгах

**Developer Tools → Application → Local Storage:**
- `http://localhost:3000` сонгоно уу
- Дараах key-нүүд байгаа эсэхийг шалгана уу:
  - `token`
  - `user`
  - `userType`
  - `adminType`

**Хэрэв байхгүй бол:**
- Нэвтрэлт амжилтгүй болсон
- Console дээр алдааг шалгана уу

#### 5️⃣ Dashboard хуудас байгаа эсэхийг шалгах

Шууд очиж үзнэ үү:
```
http://localhost:3000/admin/dashboard
```

**Хэрэв 404 бол:**
- Dashboard хуудас үүсээгүй байна
- `app/admin/dashboard/page.tsx` файл байгаа эсэхийг шалгана уу

## Түгээмэл асуудлууд:

### ❌ "Хүсэлт хэт удаан байна"
**Шалтгаан:** API 10 секундээс удаан хариулж байна
**Шийдэл:**
- Сервер ажиллаж байгаа эсэхийг шалгана уу
- Database холболт зөв эсэхийг шалгана уу
- `node scripts/test-login.js` ажиллуулна уу

### ❌ "Нэвтрэх явцад алдаа гарлаа"
**Шалтгаан:** API алдаа буцаасан
**Шийдэл:**
- Browser console дээр дэлгэрэнгүй алдааг харна уу
- Terminal дээр сервер log-ийг шалгана уу

### ❌ Нэвтэрсэн ч dashboard харагдахгүй
**Шалтгаан:** Routing асуудал
**Шийдэл:**
- Browser console дээр "🚀 Чиглүүлж байна" log байгаа эсэхийг шалгана уу
- Шууд `/admin/dashboard` руу очиж үзнэ үү
- Сервер дахин эхлүүлнэ үү

