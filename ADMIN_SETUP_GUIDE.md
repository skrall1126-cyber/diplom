# 🔐 Админ хэрэглэгч үүсгэх заавар

## Алхам 1: Supabase руу нэвтрэх

1. Энэ холбоос руу орно уу: https://supabase.com/dashboard
2. Өөрийн бүртгэлээр нэвтэрнэ үү
3. **indra-cyber-home** төслөө сонгоно уу

## Алхам 2: SQL Editor руу орох

1. Зүүн талын цэснээс **SQL Editor** дээр дарна уу
2. Эсвэл энэ шууд холбоос руу орно уу: https://supabase.com/dashboard/project/xdywicnxzgtstrslhvcu/sql

## Алхам 3: SQL кодыг хуулах

1. `scripts/insert-users.sql` файлыг нээнэ үү
2. Бүх агуулгыг хуулна уу (Ctrl+A, Ctrl+C)

## Алхам 4: SQL-ийг ажиллуулах

1. SQL Editor дээр хуулсан кодоо буулгана уу (Ctrl+V)
2. Баруун доод буланд байгаа **Run** товч дээр дарна уу
3. Амжилттай бол "Success. No rows returned" гэсэн мессеж гарна

## Алхам 5: Нэвтрэх

Одоо та дараах хэрэглэгчдээр нэвтэрч болно:

### 🔴 Супер Админ (Бүрэн эрхтэй)
- **Хэрэглэгчийн нэр:** `admin`
- **Нууц үг:** `admin123`
- **Эрх:** Системийн бүх хэсгийг удирдах

### 📚 Сургалтын Албаны Админ
- **Хэрэглэгчийн нэр:** `training_admin`
- **Нууц үг:** `training123`
- **Эрх:** Сургалтын хэсгийг удирдах

### 💰 Санхүүгийн Албаны Админ
- **Хэрэглэгчийн нэр:** `finance_admin`
- **Нууц үг:** `finance123`
- **Эрх:** Санхүүгийн хэсгийг удирдах

### 👨‍🏫 Багш
- **Хэрэглэгчийн нэр:** `teacher1`
- **Нууц үг:** `teacher123`

### 👨‍🎓 Оюутан
- **Хэрэглэгчийн нэр:** `student1`
- **Нууц үг:** `student123`

## ⚠️ Анхааруулга

"Invalid API key" алдаа гарч байсан шалтгаан:
- Өмнө нь хэрэглэгчид өгөгдлийн санд байхгүй байсан
- Одоо дээрх SQL-ийг ажиллуулснаар бүх хэрэглэгчид үүсэх болно

## 🔒 Аюулгүй байдал

Анхны нэвтрэлтийн дараа:
1. Супер админ хэрэглэгчийн нууц үгийг заавал солино уу
2. Тест хэрэглэгчдийг (teacher1, student1) устгаж болно
3. Бодит хэрэглэгчдийг системээр дамжуулан үүсгэнэ үү

## 🆘 Асуудал гарвал

Хэрэв "Invalid API key" алдаа гарсаар байвал:
1. `.env.local` файл дахь `SUPABASE_SERVICE_ROLE_KEY` зөв эсэхийг шалгана уу
2. Supabase төсөл идэвхтэй эсэхийг шалгана уу
3. SQL амжилттай ажилласан эсэхийг SQL Editor-ийн түүхээс шалгана уу
